##Weather app

###Overview of the Project
Check out the current weather for a few major cities. Navigate to the side bar to see current stats or click on a certain city to see the 5 day forcast.

##Technologies Used
* HTML5
* CSS3
* AngularJS
* JavaScript
* Google Maps API
* Open Weather API
